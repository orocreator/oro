## Commands

### Install (safe mode)

First-time setup from committed lockfile:

```bash
pnpm install --frozen-lockfile
```

This installs existing dependencies without modifying the lockfile.

### Development

```bash
pnpm dev
```

### Lint

```bash
pnpm lint
```

### Type Check

```bash
pnpm typecheck
```

### Security Audit

```bash
pnpm audit
```

Run weekly (minimum) to check for known vulnerabilities.
